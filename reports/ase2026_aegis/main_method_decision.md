# Main Method Decision

## Evidence Review
- **AEGIS hierarchy** – `reports/ase2026_aegis/table_main.csv` shows the best hierarchical variant (`aegis_full`) reaching only 0.625 success with negative reward (‑5.86). Stage‑B pretraining (`results/aegis_rl/aegis_reduced_run/reports/stage_b_summary.json`) never converged, so the manager oscillates and trails even the heuristic baseline.
- **TARL (teacher-guided residual learning)** – `results/aegis_rl/tarl_table_main.csv` again reports 1.0 success with 0 overrides. The policy simply mirrors the teacher, offering no empirical lift over imitation.
- **Legacy STRIDE** – Teacher-only STRIDE (`stride_without_uncertainty_features`) remains at 0.963 success with 0 overrides (`reports/ase2026_aegis/stride_table_main.csv`). Residual variants still regress to ~0.78 success with override regret ≈ win rate (`reports/ase2026_aegis/stride_table_ablation.csv`).
- **Counterfactual STRIDE** – The new branch-rollout dataset (`results/aegis_rl/counterfactual/summary.json`) enabled the C-STRIDE sweeps. `cstride_table_main.csv` shows that the counterfactual gate-only hybrid preserves 1.0 success with a modest 8.9% override rate and 0.82 override win rate, but every value/residual variant collapses (0.60–0.92 success, harmful override fraction ≥0.74) per `cstride_table_ablation.csv`.
- **Real Topcoder SRM repos** – The refreshed real-repo benchmark (`reports/decomposition/real_world/real_repo/summary.md`) now runs four SRM API backlog tasks end-to-end against the actual OpenAI provider (gpt‑4.1‑mini) through the repo harness, contract-aware prompts, and trace reporting. Oracle_teacher still applies the multi-file patch and passes every task (pass_rate = 1.0). The new semantic ablations (`contract_first_baseline`, `contract_first_checklist`, `contract_first`) prove that localization is solved (precision=1.0, implementation recall=1.0 across all runs) while contract coverage is the bottleneck: baseline coverage averages 0.38, checklist pushes it to 0.67, yet no strategy achieves full pass_rate because the single edited service file still violates metadata/aggregation contracts. Failure categories now read directly from the contract map (aggregation, filtering, architecture), making the remaining gap clearly semantic rather than plumbing or localization.

## Decision
The **teacher imitation baseline remains the flagship method**. Counterfactual gating provides an interpretable hybrid (rare overrides with honest win rate), yet it does not beat the teacher on success or cost, and all counterfactual value/residual models are clearly harmful. We will therefore headline the teacher / imitation policy, cite counterfactual gating as diagnostic evidence, and treat the rest of C-STRIDE as a negative-result study.
