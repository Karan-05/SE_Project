# Patch Plan

## Problem Summary
The Topcoder Community App is intermittently displaying as a single round instead of two rounds. This issue needs to be addressed to ensure accurate challenge representation.

## Plan
1. Investigate the routing logic in the application to determine why the rounds are being misrepresented.
2. Modify the relevant components to ensure that the correct number of rounds is displayed based on the challenge data.
3. Test the changes locally to ensure the issue is resolved.

## Test Plan
- Verify that challenges with two rounds display correctly.
- Check that challenges with one round still display correctly.

## Risks
- Changes may inadvertently affect other parts of the application.
- Potential for introducing new bugs during the fix.

## Validation
- Ensure that all tests pass after implementing the patch.
