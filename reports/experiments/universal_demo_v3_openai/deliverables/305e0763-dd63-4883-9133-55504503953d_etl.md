# ETL/Data Pipeline Design for Learning Space Application

## Problem Summary
The goal is to design an ETL pipeline that processes data related to user-generated content in a Learning Space application built with ReactJS. The application allows users to create learning spaces, posts, comments, and annotations. The pipeline will extract data from the source, transform it for analysis, and load it into a data warehouse for reporting and analytics.

## Source Data and Schemas

### Source Schema
1. **Users Table**
   - `user_id` (INT, PK)
   - `username` (VARCHAR)
   - `email` (VARCHAR)

2. **Learning Spaces Table**
   - `space_id` (INT, PK)
   - `title` (VARCHAR)
   - `description` (TEXT)
   - `user_id` (INT, FK)

3. **Posts Table**
   - `post_id` (INT, PK)
   - `space_id` (INT, FK)
   - `title` (VARCHAR)
   - `body` (TEXT)
   - `created_at` (TIMESTAMP)

4. **Comments Table**
   - `comment_id` (INT, PK)
   - `post_id` (INT, FK)
   - `body` (TEXT)
   - `user_id` (INT, FK)

5. **Annotations Table**
   - `annotation_id` (INT, PK)
   - `post_id` (INT, FK)
   - `highlighted_text` (TEXT)
   - `user_id` (INT, FK)
   - `comment` (TEXT)

### Destination Schema
1. **Analytics Table**
   - `analytics_id` (INT, PK)
   - `space_id` (INT)
   - `post_count` (INT)
   - `comment_count` (INT)
   - `annotation_count` (INT)

## Transformation Steps
1. **Extract Data**: Pull data from the source database tables.
2. **Transform Data**:
   - Aggregate post counts, comment counts, and annotation counts by `space_id`.
   - Clean and format text fields (e.g., trim whitespace).
3. **Load Data**: Insert the transformed data into the Analytics table.

## SQL or Code Snippets

```sql
-- Extract Data
WITH aggregated_data AS (
    SELECT 
        ls.space_id,
        COUNT(DISTINCT p.post_id) AS post_count,
        COUNT(DISTINCT c.comment_id) AS comment_count,
        COUNT(DISTINCT a.annotation_id) AS annotation_count
    FROM 
        Learning_Spaces ls
    LEFT JOIN 
        Posts p ON ls.space_id = p.space_id
    LEFT JOIN 
        Comments c ON p.post_id = c.post_id
    LEFT JOIN 
        Annotations a ON p.post_id = a.post_id
    GROUP BY 
        ls.space_id
)

-- Load Data
INSERT INTO Analytics (space_id, post_count, comment_count, annotation_count)
SELECT 
    space_id, post_count, comment_count, annotation_count
FROM 
    aggregated_data;
```
