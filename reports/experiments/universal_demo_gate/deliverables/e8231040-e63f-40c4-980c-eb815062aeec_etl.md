## Pipeline Skeleton (Python)
```python
def update_user_navigation(user_session):
    if not user_session.is_logged_in:
        navigation.show_logged_out_state()
    else:
        navigation.show_logged_in_state(user_session.user_id)
```

Extract user session data from the authentication service, transform to identify logged-out users, and load the updated user state into the frontend navigation component.
